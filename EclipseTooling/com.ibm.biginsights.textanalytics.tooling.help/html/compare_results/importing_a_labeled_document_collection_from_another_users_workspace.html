Importing a labeled data collection from the workspace of another user

Another user might already have a labeled data collection that can evaluate the quality of your extractor. Before you can use the labeled data collection, you must import it into your Eclipse workspace.

Before you begin

Ensure that you have network connectivity and access permission to the workspace of the other user.

About this task

Use the operating system copy-paste function to import the labeled data collection.

Procedure

  1. Use the operating system copy-paste function to copy the labeledCollections directory from the workspace of the other user and paste it into your workspace.
  2. Import the original data set that was used to create the labeled data collection.

    This step is required to compare an extractor run with the labeled data collection.
