Deploy an InfoSphere BigInsights application to an InfoSphere BigInsights server

Before you begin

You must have an administrator role on the relevant InfoSphere® BigInsights™ server. The application that you want to deploy must already be published to the relevant InfoSphere BigInsights server.

Procedure

  1. Select the server on which you want to deploy an application.
    Option Description
    If you created a server connection Select the server that you want to connect to from the pull down menu.
    If you did not create a server connection
    1. Click Create.
    2. Enter the full URL of the console into the URL field.
    3. Modify the server name if you want it to be different from the host name of the console.
    4. Click Test Connection to verify that the connection to the server can be established.
    5. Click Finish.
  2. Select the application that you want to deploy by using the appropriate pull down menu.
  3. Click OK. The InfoSphere BigInsights Console is displayed. If prompted for InfoSphere BigInsights Console user credentials, enter your credentials and click OK.
  4. In the Applications pane, verify that the application that you want to run is selected.
  5. Click Deploy on the Applications pane to display the Deploy Application window for the highlighted application.
  6. Enter the requested information. In particular, provide the security information for who can run the application.
  7. Click Deploy in the Deploy Application window. The deployed application shows as deployed in the Application pane.
