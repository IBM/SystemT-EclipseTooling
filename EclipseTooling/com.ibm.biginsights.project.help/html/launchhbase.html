Starting an HBase shell to run commands

You can start an HBase shell from the InfoSphere® BigInsights™ Tools for Eclipse to run HBase commands.

About this task

See the InfoSphere BigInsights installation instructions to get started.

To run the HBase shell as the user that registered the InfoSphere BigInsights server:
  • Ensure that the user of the operating system from which you start the HBase shell is the same as the BigInsights user.
  • Configure the operating system user ID to run on behalf of the BigInsights user. For more information, see Configuring proxy users for your Hadoop File System.

Procedure

  1. Select a server to run a command from in the BigInsights server view.
  2. Right-click the server and select Open HBase Shell. An HBase shell window opens in the console view.
  3. Type your HBase command, and click Enter to run the command.
  4. To close the HBase shell, click Terminate in the Console view.
