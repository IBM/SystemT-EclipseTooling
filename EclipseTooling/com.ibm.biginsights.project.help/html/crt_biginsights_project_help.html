Creating a BigInsights project

You must create an InfoSphere® BigInsights™ project before you can create applications, run queries, or analyze text.

Procedure

  1. Click File > New > Other.
  2. Expand the BigInsights tree, and click BigInsights Project, then click Next.
  3. Type the name in the Project name field, click Next, then click Finish.

    For more information, see Adding BigInsights nature to a project.
