Create an InfoSphere® BigInsights™ Java project

Procedure

  1. Select a project name and location.
  2. Specify a JRE for your project.

    Select the execution environment, a project specific JRE or the default JRE. Click Configure JREs to add, remove, or edit JRE definitions.

  3. Specify your project layout.

    Click Configure default to specify the build path entries.

  4. Select your working sets.
  5. Click Next to define your Java build settings.
  6. Click Finish to create the Java project.
