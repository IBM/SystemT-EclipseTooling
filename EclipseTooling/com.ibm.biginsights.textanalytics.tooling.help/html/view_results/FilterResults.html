Filtering results

Analyze the specific results you want by filtering results in the Annotation Explorer.

About this task

Results are displayed in the Annotation Explorer in five columns on which the filters are based:
Input Document
The file name of the data collection, which can be the actual name of the file or an internally used identifier.
Left Context
The 50 characters immediately before the span value.
Span Attribute Value
The actual text of the span.
Right Context
The first 50 characters that follow the span value.
Span Attribute Name
The name of the output view and the field name of the span, which is separated by a period.

Procedure

  1. Click the Select Filters for the Annotation Explorer menu on the Annotation Explorer toolbar. Click Show hide filter view, which is a toggle switch.
  2. In the Filter Criteria dialog, click Configure Filter to specify the optional filter criteria for every column on which you want to filter on.
    Option Description
    Span Attribute Name

    Highlight the span columns that you want to display. Click Add and then click OK.

    Span Attribute Value

    Enter the text pattern that you want to match, and click OK.

    Input Document

    Select the documents that you want to display. Click Addand then click OK.

    Left Context

    Enter the text pattern that you want to match, and click OK.

    Right Context

    Enter the text pattern that you want to match, and click OK.

  3. Click the Select Filters for the Annotation Explorer menu on the Annotation Explorer toolbar, and click Show hide filter view. The Filter Criteria dialog closes. Filter criteria is saved across sessions, so there is no need to reenter the criteria the next time you start the InfoSphere® BigInsights™ Eclipse Tools. Of course, you can remove existing criteria for any column by clicking Clear Filter.
