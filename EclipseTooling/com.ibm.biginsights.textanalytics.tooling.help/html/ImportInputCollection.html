Importing a data collection

To run your application successfully, you must import a data collection into your InfoSphere® BigInsights™ project.

Procedure

  1. From the Package Explorer, right-click the project, and click Import.
  2. In the Import wizard, click General > File System, and click Next.
  3. Click Browse to select the directory that contains the data collection.
  4. Click Browse to select the folder into which to import the collection.
  5. Click Finish.
