Explain Module View

The Explain Module view, displays the module metadata and the compiled form of the extractor. This feature is equivalent to the Explain Plan feature of a commercial relational database.

Procedure

  1. Click Window > Other > BigInsights > Explain Module to open the Explain Module View.
  2. Select the Text Analytics Module (.tam ) file location by clicking either Browse Workspace or Browse File System to view the contents of the module.
