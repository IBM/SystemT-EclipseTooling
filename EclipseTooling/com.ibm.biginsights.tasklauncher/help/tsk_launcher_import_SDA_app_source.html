Import application source for social media analytics

Before you begin

Download and install the IBM® Accelerator for Social Data Analytics.

Procedure

  1. Select the InfoSphere® BigInsights™ server from which you want to import a project.
    Option Description
    If you created a server connection Select the server that you want to connect to from the pull down menu
    If you did not create a server connection
    1. Click Create.
    2. Enter the full URL of your console into the URL field.
    3. Modify the server name if you want it to be different from the host name of the console.
    4. Click Test Connection to verify that the connection to the server can be established.
    5. Click Finish.
  2. Select an Eclipse project, then click OK to import the project.
