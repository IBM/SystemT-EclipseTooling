Result Editor

By using the Result Editor you can visualize results in the input data or data collection. For each attribute type that you select in the Result tree view, the Result Editor shows all of the annotations. The Result Editor is opened whenever the Result Tree is opened.

Color Highlighting

Because each attribute type that is displayed in the Result Editor is highlighted in a different color, you can easily visualize in the input document all of the annotations for a particular annotation type. Moreover, because the Result Editor displays all of the documents in the data collection, you can compare the annotations across the data collection.
