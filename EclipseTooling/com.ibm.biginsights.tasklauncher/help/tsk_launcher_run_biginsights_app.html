Run and monitor an InfoSphere BigInsights application

Procedure

  1. Select the server on which you want to run an application.
    Option Description
    If you created a server connection Select the server that you want to connect to from the pull down menu.
    If you did not create a server connection
    1. Click Create.
    2. Enter the full URL of the console into the URL field.
    3. Modify the server name if you want it to be different from the host name of the console.
    4. Click Test Connection to verify that the connection to the server can be established.
    5. Click Finish.
  2. Select the application that you want to run by using the appropriate pull down menu.
  3. Click OK. The InfoSphere® BigInsights™ Console is displayed. If prompted for InfoSphere BigInsights Console user credentials, enter your credentials and click OK.
  4. In the left pane of the Execute task of the Applications tab, verify that the application that you want to run is selected. The upper right pane displays the information about the selected application.
  5. Review and optionally modify the application description, run parameters, and other settings. After the information is correct, click Run.

Results

You can now monitor the run progress of the selected application in the Application History pane.
