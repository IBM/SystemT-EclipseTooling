Creating a file to contain Jaql

Procedure

  1. Enter a name for the file and specify the location in which the file will be saved. While you type the name of the file, the wizard checks if a file with the same name already exists in the specified location. If you specify a file extension, it must be .jaql. If you do not specify a file extension, .jaql is automatically added.
  2. Click Finish. A file with a simple comment is created in your selected location.
