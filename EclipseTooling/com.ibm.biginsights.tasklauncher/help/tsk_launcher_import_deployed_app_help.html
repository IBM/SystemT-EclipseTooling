Importing a deployed application from the cluster

Procedure

  1. Select a BigInsights server and BigInsights application, and click OK.
  2. Select a folder into which the application will be imported.

    An app.zip file is placed in the specified folder and uncompressed. A folder named app , which includes the uncompressed content, appears in the directory you specified.

  3. Optional: If you want to re-deploy this application to the cluster:
    1. Select the app folder.
    2. From the context menu, click Export.
    3. Under the BigInsights category, select Application Publish.
