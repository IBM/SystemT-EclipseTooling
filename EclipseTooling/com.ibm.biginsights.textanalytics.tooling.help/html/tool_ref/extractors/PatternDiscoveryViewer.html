Pattern Discovery View

The Pattern Discovery View opens first when Pattern Discovery completes and displays results from discovering patterns in text input. It consists of brightly colored circles that represent different groups and their semantic patterns. Patterns that are too long to fit in a circle are displayed in the tooltip. The larger the group size, the larger the corresponding circle. By default, the top ten largest groups are displayed.

The label of each circle consists of its Common Semantic Pattern, a list of keywords and entity types that best describe a group of similar input contexts, and the size of the underlying group of input contexts that are summarized by this pattern.

Manipulate the visualization using the following actions:

Move
Click and drag to reposition the visualization in the view.
Zoom
Right-click and drag to zoom in and out.
View details
  • Hover over a circle to display its tooltip. Select a circle to visualize the details about the semantic pattern of the group in the Pattern Context View.
  • Right-click and select See History to show the history of the pattern, including sequences that were disregarded by the pattern discovery algorithm because they were highly correlated with this pattern.
Filter
Click the Filter icon at the top of the view to filter groups by size. The 10 largest groups are shown by default.

Actions that are available from the toolbar:

Set minimum sequence length
Displays an adjustable slider. Adjust the value for the minimum sequence length. For information about minimum sequence length, see Pattern-discovery configuration: Advanced settings
Note: Adjusting the value automatically causes pattern discovery to run again.
Export all patterns
Exports all of the discovered patterns to a CSV file.
Export displayed patterns
Exports only the displayed patterns to a CSV file.
The CSV file contains the following fields:
  • Group Size
  • Common Pattern
  • Semantic Pattern
  • Context
  • Data File
