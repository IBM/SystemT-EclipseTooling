Creating an InfoSphere BigInsights server connection

Connect to the InfoSphere® BigInsights™ server to enhance your Eclipse development environment with capabilities for developing and deploying applications to the InfoSphere BigInsights server and writing programs using Java MapReduce, JAQL, Pig and Hive. They also support developing Text Analytics programs, such as extractors, that run on IBM InfoSphere BigInsights.

Procedure

  1. Open the InfoSphere BigInsights server view by clicking Window > Show View > Other.
  2. Under the BigInsights category, select BigInsights Servers.
  3. In theBigInsights Servers view, right-click BigInsights Servers, and click New. The Create a new BigInsights server window opens.
  4. Copy the URL of your InfoSphere BigInsights Console into the URL field. The name of the server defaults to the host name of the InfoSphere BigInsights Console, but you can change it.
  5. If your server is set up for security, enter your credentials in the User ID and Password fields.
  6. Optional: Click Test connection to verify that you can connect to the server.
  7. Click Finish.
