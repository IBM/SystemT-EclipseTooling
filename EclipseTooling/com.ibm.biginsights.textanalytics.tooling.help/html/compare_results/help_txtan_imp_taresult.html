Importing Text Analytics results

You can import the JSON records that result from running an extractor into an InfoSphere® BigInsights™ project.

Before you begin

You must have access to Text Analytics results.

About this task

The ability to import results from previous extractors makes already produced records available for examination as if you had labeled the documents yourself. You can share any available results within the InfoSphere BigInsights Eclipse environment.

Procedure

  1. In the InfoSphere BigInsights Eclipse tools, click File > Import > BigInsights > Text Analytics Results.
  2. In the Text Analytics result field of the Import Text Analytics Results window, browse to select a Text Analytics result JSON file.
  3. In the Import into project field, browse and select a project as the destination of the results.
  4. Optional: Specify the Open results in Annotation Explorer when import completes check box to open the Annotation Explorer view with the results when the import is complete.
  5. Click Finish.

Results

The JSON file is converted to a folder of the same name as the original JSON file inside the result folder of the project. Inside that folder are *.strf text files.

The Annotation Explorer view contains the results, showing the Left Context and Right Context columns and the Span Attribute Value for each of those columns. For more information about the view, see Annotation Explorer.
