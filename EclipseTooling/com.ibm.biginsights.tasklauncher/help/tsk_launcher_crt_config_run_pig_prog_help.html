Create a run configuration for Pig scripts

Procedure

  1. Specify a name for the configuration.
  2. Specify the Pig Main Settings.
    1. Select a project name, and the Pig script to use in the configuration.
    2. Select an Exection mode.
    Option Description
    Select Cluster Select a BigInsights server from the drop-down menu.
    Select Local Run the configuration locally.
  3. Specify the parameter names and values.

Steps 3 - 7 are optional. In most cases, you can use the default settings and skip to step 8.

  1. Optional: In the Arguments tab, select the variables for the program and VM arguments, and specify the working directory.
  2. Optional: Define the runtime JRE and the execution environment in the JRE tab.
  3. Optional: In the Classpath tab, organize and remove items, add projects and JAR files, define advanced options (such as folders, classpath variables, or variable strings), or restore the default settings.
  4. Optional: Define your environment variables in the Environment tab.
  5. Optional: In the Common tab, specify file sharing, a favorites menu, encoding, standard input and output, and appearance.
  6. Click Apply to change settings or Revert to restore the default settings.
  7. Click Run to launch the configuration.
