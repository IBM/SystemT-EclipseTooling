Comparing the results of one extractor run to the results of another run

Comparing the results of one extractor run to the results of another run can be useful when you are testing an extractor. Each time a launch configuration is run, a new subdirectory named result-system-timestamp is saved in the result directory of the project.

Procedure

  1. Right-click the first result folder in the Project Explorer on which you want to perform comparative analysis.
  2. Click Compare Text Analytics Result With … > Other Analysis

    All result folders from the project are listed.

  3. From the list of result folders, select the second result folder that you want to analyze, and click OK.
  4. Compare the results side-by-side in the Annotation Difference Viewer.

    See Annotation Difference Viewer for an explanation of the result differences.
