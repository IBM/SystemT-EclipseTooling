Create a BigInsights server connection

Procedure

  1. Enter the full URL of your console into the URL field. The URL must contain the complete web address, such as my.server.com:port.
    my.server.com
    The name of the server where your console is running.
    port
    The port number for your console.
  2. Optional: Modify the server name if you want it to be different from the host name of the console.
  3. If your server is configured for security, enter information for the User ID and Password fields.
  4. Click Test connection to verify that the connection to the server can be established. If the connection is successful, a window opens to verify that the system connected to the web server.
  5. Click Finish to close the BigInsights server connection window.

Results

Your server is listed in the BigInsights Servers view.
