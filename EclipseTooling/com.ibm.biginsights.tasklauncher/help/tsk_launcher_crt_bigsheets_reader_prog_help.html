Create a BigSheets reader

Procedure

  1. Create a BigSheets reader class. Select your source folder, and select or specify the package. Create a name for the BigSheets reader, and select a Super class from the drop-down menu. Select the types of reader keys and values.
    1. Optional: Add or select any interfaces to use.
  2. Click Finish to create the reader.

Results

The BigSheets reader opens in a new window.
