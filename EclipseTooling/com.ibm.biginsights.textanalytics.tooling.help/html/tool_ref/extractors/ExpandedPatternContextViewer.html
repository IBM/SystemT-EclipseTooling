Expanded Pattern Context View

The Expanded Pattern Context View displays results from discovering patterns in text input.

To help with rule identification, the view displays text snippets that surround the context value of the subgroup in the input data set. By default, the surrounding snippet consists of 25 characters to the left and right of the context. A different snippet can be specified in the Snippet Field Name field of the Advanced Configuration page.

The four snippets that surround 'please call me at' from the Pattern Discovery View in our example are shown in the Table 1 (a custom snippet of 10 tokens to the left and right of the input context is used). This view becomes useful when you notice that even though the four input contexts are the same, their surrounding snippet is different.
  • Context: please call me at
  • Semantic Pattern: call;at
  • Common Pattern: at
