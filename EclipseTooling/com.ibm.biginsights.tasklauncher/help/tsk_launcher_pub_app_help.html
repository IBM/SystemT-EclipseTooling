Publish an InfoSphere BigInsights application

Procedure

  1. On the Location page, select an InfoSphere® BigInsights™ project.
  2. Select the server on which you want to publish an application.
    Option Description
    If you created a server connection Select the server that you want to connect to from the pull down menu.
    If you did not create a server connection
    1. Click Create.
    2. Enter the full URL of the console into the URL field.
    3. Modify the server name if you want it to be different from the host name of the console.
    4. Click Test Connection to verify that the connection to the server can be established.
    5. Click Finish.
  3. Click Next.
  4. On the Application page, select whether you want to create an application or replace an existing one. Provide more information about the application as needed. Click Next.
  5. On the Type page, select the type of application you want to create. Depending on what type you select, the navigation and content varies through the:
    1. Workflow page,
    2. Text Analytics page,
    3. BigSheets page,
    4. Parameters page.
    For more information, see Creating BigInsights Applications.
  6. On the Publish page, preview the application archive that you want to create and publish.
  7. Click Finish to publish the application to the InfoSphere BigInsights cluster.
