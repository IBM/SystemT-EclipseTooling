Pattern Context View

The Pattern Context View displays results from discovering patterns in text input.

Each group can have one or more subgroups, and each subgroup can have a different semantic pattern. The semantic patterns of the subgroups are similar enough that they can be grouped under the common semantic pattern of the group.

The view displays information about a subgroup in a tabular format with three columns:
Context
The content of the input contexts that are represented by the subgroup.
Size
The number of input contexts that are represented by the subgroup (the same context value can occur multiple times in the input).
Semantic Pattern
The semantic pattern of the subgroup.

The group has two subgroups with different semantic patterns at and call;at . Since the two patterns are similar to each other, they are joined into a single group under the common pattern at . The size column indicates how many times a particular context occurred in the input. Select a subgroup to open the Expanded Pattern Context View that displays the surrounding text of the corresponding context in the input data set.
