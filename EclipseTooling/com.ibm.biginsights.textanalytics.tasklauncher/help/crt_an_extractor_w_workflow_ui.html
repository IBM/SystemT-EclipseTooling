Create an extractor by using the Text Analytics Workflow perspective

Procedure

  1. Create a BigInsights project.
    1. Specify a project name. Click Use default location, or specify a new location.
    2. Select a BigInsights library version to use from the drop-down menu.
    3. Click Finish. Your project will appear in the Extraction Plan tab and the Text Analytics Workflow perspective opens in the Extraction Tasks tab.
  2. In the Text Analytics Workflow perspective, create your extractor. Complete the steps in the Text Analytics Workflow perspective.
