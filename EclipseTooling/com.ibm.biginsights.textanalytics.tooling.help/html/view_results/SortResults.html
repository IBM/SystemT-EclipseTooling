Sorting results

You can sort results in the Annotation Explorer or Result Table View.

About this task

Results are displayed in the Annotation Explorer in five columns:
Input Document
The file name of the data collection, which can be the actual file name or an internally used identifier.
Left Context
The 50 characters that precede the span value.
Span Attribute Value
The actual text of the span.
Right Context
The 50 characters that follow the span value.
Span Attribute Name
The name of the output view and the name of the span separated by a period.

Procedure

  1. Click a column header. The rows are sorted according to the entries in that column.
  2. Optional: Click a second column header. The rows are sorted first by the second column header that you clicked, and then the rows are sorted by the first column header that you clicked.
