Creating a BigInsights project

Procedure

  1. Enter a name for your project in the Project name field.
  2. Check Use default location or click Browse to select the location directory, and then click Next.
  3. Select the BigInsights library version that you want to use from the pull down list.
  4. Click Finish.

Results

Your project is listed in the Project Explorer window.

What to do next

You can add a BigInsights nature to your project so that you can create MapReduce applications, run Jaql or Hive queries, or run Text Analytics. For more information, see Adding BigInsights nature to a project.
