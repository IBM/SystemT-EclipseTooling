Monitor InfoSphere BigInsights jobs running in a cluster

Procedure

  1. Set the values for connecting to the InfoSphere® BigInsights™ Console
    1. Enter the URL and name of the InfoSphere BigInsights server.
    2. Enter your user name and password, and click Test Connection. If your connection in successful, click Next.
  2. Select the server that you want to monitor.
    Option Description
    If you created a server connection Select the server that you want to connect to from the pull down menu
    If you did not create a server connection
    1. Click Create.
    2. Enter the full URL of your console into the URL field.
    3. Modify the server name if you want it to be different from the host name of the console.
    4. Click Test Connection to verify that the connection to the server can be established.
    5. Click Finish.
  3. Click OK.
  4. On the Job Details page, select a job to see details. You can apply filters to only show jobs that match a specific criteria. When you click on the job, you can see the tasks counters, and configurations for the job.

Results

The Job Details window opens and lists details about each job that exists on your server. You use this window to monitor BigInsights jobs that are running in your cluster. You can apply filters to show only the jobs that match a specific criteria.

By clicking on a job, you can view the tasks, counters, and configuration for the job.
